# PWC Job Simulation

Project : Power BI Dashboard & KPIs-Metrics

Objective : To provide useful dashboards which show KPIs & Metrics related to given three tasks :

1) Call Center : The Dashboards show Customer Satisfaction, Agent Statistics, Calls Resolved & Answered, Total Calls per month.
2) Customer Retention : Gives the insights of Customer Demographics, Account Information, Internet Services, and Risk Analysis.
3) Diversity & Inclusion : Shows Employee Demographics, Hiring Rate, Turnover Rate, Gender based Employee information.
